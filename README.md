# Write-ups de Challenges

Bienvenue sur mon dépôt de solutions de challenges en cybersécurité ! Ce repo me sert de base de connaissances personnelle et permet de documenter ma progression et ma méthodologie sur différents challenges cyber.

**Auteur :** `Albinator19`

---

## Structure du Dépôt

Les solutions sont organisées par plateforme et par thématique :

* **[OverTheWire](https://overthewire.org/wargames/)**
    * `Bandit` : Fondamentaux Linux, SSH, Scripting bash et Git.
* **[HackTheBox](https://app.hackthebox.com/)**
    * `Three` : Fondamentaux AWS, S3 Bucket, ACL, Subdomain Enumeration.
    * `Archetype` : Windows, SMB, Microsoft SQL Server, winpeas.exe, nc64.exe.
    * `Oopsie` : IDOR, Cookie, Path Traversal, SUID, Upload.
    * `Vaccine` : FTP, zip2john, JohnTheRipper, SQLMAP, vi.
    * `Unified` : CVE (Log4Shell), Exploit, JNDI, LDAP, MongoDB.
    * `Silentium` : API, Flowise, Gogs, symlink, CVE-2025-58434, CVE-2025-59528, CVE-2025-8110.
    * `Cap` : IDOR, Wireshark, FTP, Capability.
